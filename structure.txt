│
├─ generate_cert.py
│  └─ 기능: `cert.pem`과 `key.pem` 같은 자체 서명된 SSL/TLS 인증서와 개인 키를 생성하는 Python 스크립트입니다.
│
├─ index.html
│  └─ 기능: 웹 애플리케이션의 메인 페이지입니다. 전체적인 UI 구조(헤더, 지도, 통계, 설정, 알림창 등)를 정의하고, CSS와 JavaScript 파일을 로드합니다.
│
├─ key.pem
│  └─ 기능: SSL/TLS 개인 키 파일입니다. HTTPS 통신을 암호화하는 데 사용됩니다.
│
├─ server.py
│  └─ 기능: Flask 기반의 백엔드 서버입니다. 웹 페이지를 제공하고, 웹소켓 통신을 통해 클라이언트로부터 실시간 오디오 데이터를 수신합니다. 수신된 오디오는 TensorFlow의 YAMNet 모델로 분석하여 경적, 사이렌 등 위험 소리를 감지하고, 감지 시 클라이언트에 알림을 보냅니다.
│
├─ structure.txt
│  └─ 기능: 현재와 같이 프로젝트 파일 구조와 각 파일의 역할을 설명하는 텍스트 파일입니다.
│
├─ css
│  └─ style.css
│     └─ 기능: 웹 애플리케이션의 모든 시각적 스타일을 정의합니다. 레이아웃, 색상, 폰트, 다크 모드, 애니메이션 등 UI의 전체적인 디자인을 담당합니다.
│
├─ images
│  ├─ bike-icon.svg
│  │  └─ 기능: 통계 페이지 등에서 사용되는 자전거 아이콘 이미지입니다.
│  ├─ logo.png
│  │  └─ 기능: 애플리케이션의 로고 이미지입니다.
│  └─ logo1.png
│     └─ 기능: 애플리케이션의 또 다른 로고 이미지입니다.
│
├─ js
│  └─ script.js
│     └─ 기능: 웹 애플리케이션의 핵심 프론트엔드 로직을 담당합니다. SafeRideApp 클래스를 통해 UI 상호작용, 서버와의 웹소켓 통신, 마이크 오디오 캡처 및 전송, 실시간 위험 경고 표시, 주행 상태(시작, 정지, 일시정지) 관리, 통계 계산, 카카오맵 연동 등 애플리케이션의 모든 동적 기능을 제어합니다.
│
├─ sounds
│  └─ warning.mp3
│     └─ 기능: 서버로부터 위험 감지 알림을 받았을 때 재생되는 경고음 파일입니다.
│
└─ __pycache__
   └─ 기능: Python 인터프리터가 생성하는 캐시 파일들이 저장되는 폴더입니다. 애플리케이션 실행 속도를 높여주며, 직접 수정할 필요는 없습니다.
